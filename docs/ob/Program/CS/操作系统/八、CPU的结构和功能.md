## 八、CPU的结构和功能

### 1、CPU的结构

#### 一、CPU的功能

*   控制器的功能-
    取指令 指令控制-
    分析指令-
    执行指令，发出各种操作命令 操作控制-
    控制程序输入及结果的输出 时间控制-
    总线管理 处理中断-
    处理异常情况和特殊请求
*   运算器的功能-
    实现算术运算和逻辑运算
